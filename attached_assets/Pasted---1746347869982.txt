Я создам подробный промт для разработки компьютерного приложения по изучению методов машинного обучения с практическим применением в области пожарной безопасности.
Промт для разработки образовательного приложения "ML FireSafe"
Общее описание
Необходимо разработать интерактивное образовательное приложение "ML FireSafe", направленное на изучение методов машинного обучения с практическим применением в области пожарной безопасности. Приложение должно представлять собой полноценную обучающую платформу с системой тестирования, отслеживанием прогресса и современным удобным интерфейсом.
Структура приложения
1. Система авторизации и управления пользователями

Регистрация нового пользователя (имя, электронная почта, пароль)
Авторизация существующих пользователей
Восстановление пароля
Личный кабинет с отображением прогресса обучения
Различные роли пользователей (студент, преподаватель, администратор)

2. База данных

Хранение данных пользователей (профили, прогресс обучения)
Сохранение результатов тестирования
Учебные материалы и медиаконтент
Наборы данных для практических занятий
Статистика использования приложения
